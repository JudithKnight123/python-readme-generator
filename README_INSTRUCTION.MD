# Python README Generator

## Description

This application is a command-line interface tool designed to streamline documentation. As a developer, I created this script to automatically generate professional `README.md` files for GitHub repositories, saving time and ensuring consistency across projects.

## Key Learnings

- Python scripting and project architecture
- Implementing classes and modules for clean, reusable code.
- Utilizing `PyInquirer` for interactive CLI prompts.
- Managing Python dependencies and virtual environments.

## Installation

To run this application on a mac, ensure you have Python3 installed and follow these steps:

Clone this repository to your local machine using the command line

git clone 'https://github.com/JudithKnight123/python-readme-generator.git'

## About This Project

This is a helpful tool that makes it easy to create a professional description file for your projects. Instead of writing everything from scratch, you simply answer a few questions in the terminal, and the application builds the file for you.

## How to Get Started

To use this tool, please follow these steps on your computer:

1. Open the project folder in Visual Studio Code.
2. Right-click on the folder name in the file explorer sidebar and select "Open in Integrated Terminal."
3. Set up the virtual environment by typing:
   python3 -m venv .venv
4. Install the requirements by typing:
   pip3 install -r requirements.txt
5. Run the application by typing:
   python3 main.py (mac) or python main.py (windows)
6. Follow the prompts by using your arrow keys to select your choices and the Enter key to submit your answers.
